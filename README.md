# Lumao-Game-Project-Scripts
Core C# scripts and game logic for 'Lumao', a Unity-based racing and taxi simulation game. Includes AI car controllers, fuel systems, and UI management.

This repository contains the essential C# scripts and game logic for Lumao, a Unity-based racing and taxi simulation game.

Key Technical Features:
AI Car Controller: Advanced logic for non-player vehicles.

Player Vehicle Physics: Custom car controller and fuel system.

Game Management: Systems for race timers, tutorials, and collectables.

UI Implementation: Flexible UI management for menus and HUDs.

Note: This repository focuses on the technical scripts. Assets and environment files are not included due to size constraints.
